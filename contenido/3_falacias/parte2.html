Déficit del Gobierno
  1. Los gastos se distribuyen de acuerdo a cada meta.
  2. No deben desviar los bienes y servicios que son asignados para cada habitante.
  3. Si la situación lo amerita se debe imprimir dinero.
  4. Se debe “saber a quién le falta y a quién le sobra”.
El Desempleo
  1. Es una bendición, siendo indicativo que con la mano de obra que se tiene se están cumpliendo las metas.
  2. Si las metas se están cumpliendo no es necesario crear tantas fuentes de trabajo.
  3. Una buena opción es reducir las jornadas laborales y así curar el problema del excedente.
Modernización Tecnológica
  1. La tecnología importada está hecha para solventar los problemas de donde proviene.
  2. Toda tecnología importada implica una atadura.
  3. Se debe contar con una política científica-tecnológica propia y autónoma.
Desarrollo
  1. El desarrollo debe ser relativo al cumplimiento de las metas planteadas.
  2. Se debe aspirar al crecimiento de autónomo, no ser como otros países.
